# nightmare_web
Nightmare is a front-end web project presenting a fictional energy drink brand inspired by nightlife and gaming culture. Built with HTML, CSS and JavaScript, the website showcases product flavors, ingredients and cocktail ideas through a cyberpunk-style landing page with a strong visual identity.


Nightmare is a front-end web development project that presents the concept of a fictional energy drink brand targeted at nightlife and gaming communities. 

The website acts as a promotional landing page that introduces three different product flavors, highlights their ingredients, and suggests cocktail combinations. The design focuses on a dark cyberpunk aesthetic with neon elements to reflect the brand’s identity and appeal to a young audience interested in nightlife, electronic music, and gaming culture.

This project was developed using only HTML, CSS, and JavaScript, with the goal of demonstrating front-end development techniques, responsive layout design, and digital branding principles.

The website structure follows a marketing-focused layout including hero sections, product descriptions, ingredient highlights, and promotional content designed to simulate a real commercial landing page.
